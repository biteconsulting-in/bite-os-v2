# BITE OS V2.0 - Engineering Contract (Phase 1)

## Objective
Create and organize a clean, version-controlled repository that will become the single source of truth for the entire platform. 

> **CRITICAL:** You are not responsible for engineering decisions or code refactoring in Phase 1. 

## Responsibilities
* Create the complete repository structure.
* Import all existing production code.
* Organize all current pipelines.
* Create documentation.
* Prepare the repository for Claude to begin engineering work.
* **Do not modify any Python logic.**

## Restrictions
During Phase 1, you must NOT:
* Modify Python
* Rename files or Delete files
* Merge pipelines
* Deploy Cloud Run
* Change BigQuery, GCS, Eventarc, or IAM
* Refactor code

## Handover
When Phase 1 is complete, the repository must contain:
1. All existing production code (in `/legacy`)
2. Complete documentation
3. Complete inventory & Migration matrix
4. Empty framework ready for development
